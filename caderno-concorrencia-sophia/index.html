Guia de uso

Como manter e consultar o caderno
01

Como consultar

Use o menu superior para navegar entre as seções. Em "Concorrentes", filtre por tipo, cidade ou risco. Clique em "Ver análise" para abrir o detalhamento completo de cada empreendimento.

02

Como atualizar os dados

Abra a planilha "Cadastrode Resumo Concorrentes Sophia" no Google Drive. Edite a aba correspondente. Em seguida, execute o script de exportação para gerar os arquivos JSON atualizados na pasta data/.

03

Como adicionar concorrente

Inclua uma nova linha na aba "Cadastro_Resumo" da planilha, preenchendo todos os campos obrigatórios. Reexporte os JSONs. O caderno carregará o novo concorrente automaticamente.

04

Como exportar para PDF

Clique em "Imprimir" no menu ou use Ctrl+P (Windows) ou Cmd+P (Mac). No diálogo de impressão, selecione "Salvar como PDF". O layout é otimizado para impressão.

05

Como publicar nova versão

Substitua os arquivos JSON na pasta data/ pelos novos exportados. Se hospedado em GitHub Pages ou Netlify, faça o commit ou arraste os arquivos novos. O site atualiza automaticamente.

06

Como rodar localmente

Abra o terminal na pasta do projeto e execute npx serve . ou python -m http.server 8080. Acesse http://localhost:8080 no navegador. Não abra o index.html diretamente pelo clique duplo.

07

O que validar antes do atendimento

Verifique o campo "Prioridade de validação" de cada concorrente. Dados marcados como "A validar" ou "Pendente" não devem ser usados em conversa com cliente sem confirmação de fonte direta.

08

Como atualizar objeções

Edite a aba "Objeções_CRM" na planilha. Inclua a objeção, a leitura real, o risco, a resposta estratégica e a pergunta de avanço. Reexporte o arquivo data/objecoes.json.
